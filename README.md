# 8fate Web App

BaZi fortune-telling web application — demo deployment for dev.8fate.ai

## Tech Stack
- React + TypeScript + Vite
- React Router v6
- Zustand state management
- i18n (English + Traditional Chinese)

## Deployment
Static site deployed via Netlify to dev.8fate.ai

The `dist/` folder contains the pre-built production build.
