# Revamp (20251228b) folder scaffold

This directory is a placeholder for the 20251228b UI revamp. No production UI has been moved here yet.

## Structure rules

- `pages/` only composes components (route-level layout, no shared UI).
- `components/` houses reusable presentational UI.
- `hooks/` stores UI logic and state helpers.
- `services/` contains revamp-only API/data fetching.
- `types/` defines TypeScript types/interfaces.
- `content/` holds JSON copy/config (future i18n).
