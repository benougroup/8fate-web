# Icon system (20251228b)

Canonical source: `docs/ui-revamp/20251228b/04-icon-system.md`.

## Usage rules

* Do not recolor icons.
* Always select the correct light/dark asset by theme (yin/dark, yang/light).

## Internal name groups

* Navigation: `sun`, `year`, `lucktrend`, `me`, `mouth`, `settings`, `menu`
* System: `yinyang`
* Premium: `premium`
* Result: `upup`, `up`, `middle`, `down`, `downdown`
* Background: `background001`
* Core: `today`, `luck`, `avoid`, `protection`, `upcoming`, `energy_boost`, `future_upcoming`
